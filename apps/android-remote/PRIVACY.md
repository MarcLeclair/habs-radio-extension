# Habs Radio Remote — Privacy Policy

Habs Radio Remote does not use accounts, analytics, ads, or tracking.

## Data Collection

Habs Radio Remote does not collect, sell, or share personal data.

## Local Network

The phone remote scans your local Wi-Fi network to find the Android TV app. The remote stores the selected TV IP address and port on your device so notification controls can continue to control the TV.

## Internet Access

The phone remote sends control requests only to the TV address you select.

## Notifications

The phone remote uses notifications for quick playback controls.

## Data Sharing

Habs Radio Remote does not share user data with third parties.
